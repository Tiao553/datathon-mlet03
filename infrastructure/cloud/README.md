# Cloud Infrastructure (Terraform)

**Status:** Planned for Phase 3

This directory will contain Terraform configurations for deploying the MLOps platform to cloud providers.

## Planned Components

- **AWS/GCP Compute:** ECS/Cloud Run for API
- **Managed Databases:** RDS/Cloud SQL for Airflow/Langfuse
- **Object Storage:** S3/GCS for MLflow artifacts
- **Networking:** VPC, Load Balancers, Security Groups
- **Monitoring:** CloudWatch/Cloud Monitoring integration
